==========
User Guide
==========

**Rackspace RPC rpc-ceph**

| **Last updated:** |today|
| **Release:** |release|
|

rpc-ceph is a Rackspace RPC taylored wrapper around **ceph-ansible** to ty and
insure a constent, stable and upgradedable deployment of Ceph in a customers
environment.  

For information about how to use Kubernetes, see
`Kubernetes documentation <https://kubernetes.io/docs/home>`__.
